# Depth Vector Premium Logo Assets v3

This package uses the approved A6 app icon and H3 horizontal logo direction. It is raster-first for the premium metallic logo so it looks close to the approved concept in-app.

## Best files to use

- **App launcher icon:** `png/app_icon_A6_dark_1024.png`
- **High-res app icon master:** `png/app_icon_A6_dark_4096.png`
- **In-app horizontal header:** `transparent_png/horizontal_logo_H3_transparent_4800w.png`
- **Splash/About/Marketing header:** `png/horizontal_logo_H3_dark_4800w.png`
- **Small UI mark:** `transparent_png/dv_mark_transparent_1024w.png`

## Important
The premium metallic logo is provided as PNG/WebP. The SVG files are supporting elements only. A true vector version of the full metallic logo should be rebuilt manually in Illustrator/Figma if needed for print or infinite scaling.
